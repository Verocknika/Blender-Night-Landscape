# Night Landscape – Blender Semester Project

## Project Description
Technical documentation and 3D scene assets created for the **Modern Computer Graphics** course.  
This project focuses on a stylized low‑poly visualization of a nighttime environment, emphasizing atmospheric lighting and procedural shading.

[Blender documentation ](VM_Blend_projekt.pdf)

## Technical Workflow

### Object Modeling
- **Low‑poly vegetation** — spruce trees modeled from cylinders using `Loop Cut` and `Scale`
- **Organic shapes** — bird character modeled from a cube using the `Mirror` modifier and optimized with `Decimate`
- **Terrain** — island created from a plane using the `Knife` tool and deformed with `Mesh Transform → Randomize`
- **Environmental assets** — mushrooms using `Subdivision Surface` and stones shaped with `Displace` and Voronoi texture

### Shading and Materials
- **Starry sky** — procedural world shader using `Noise Texture` and `ColorRamp`
- **Luminescence** — `Emission Shader` applied to the moon and glowing mushrooms
- **Procedural textures** — extensive use of `Voronoi` for natural variation

### Lighting and Post‑processing
- **Atmospheric lighting** — colored `Area Lights` for soft illumination and depth
- **Compositing** — final adjustments using the `Glare` node in Fog Glow mode
